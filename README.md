# 🚀 VStream - Cloud-Based Video Streaming Platform

## 📌 Project Overview

VStream is a cloud-based video streaming platform built using HTML, CSS, and JavaScript, and deployed on Amazon Web Services (AWS) using Amazon S3. It provides educational videos across multiple technical domains with features like search and category filtering.

---

## 🌐 Live Demo

👉 Hosted on AWS S3 (Static Website Hosting)

---

## ✨ Features

* 🎥 Video streaming using HTML5 video player
* 🔍 Search functionality
* 🏷 Category-based filtering
* 💻 Clean and responsive UI
* ☁️ Cloud deployment using AWS S3

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* AWS S3 (Static Website Hosting)

---

## ☁️ Cloud Implementation

* Created an S3 bucket to store website files
* Uploaded HTML and video files
* Configured bucket policy for public access
* Enabled static website hosting
* Accessed project via public S3 endpoint

## 🚀 How to Run Locally

1. Download the project
2. Open `index.html` in a browser
3. Ensure all video files are in the same folder
